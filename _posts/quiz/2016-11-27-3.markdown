# Arrange complexity in asc order of time: O(log(n)), O(log(n!)), O(n^2), O(n) ?

log (n!) = log ( (n)\*(n-1)\*(n-2)\*...(1) )

log (n!) = log (n) + log (n-1) + log (n-2) + log (1)

log (n!) = nlog (n)

Therefore:

O(log(n)) < O(n) < O(nlog(n)) < O(n^2)
