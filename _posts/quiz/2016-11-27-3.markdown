  question: "Arrange complexity in asc order of time: O(log(n)), O(log(n!)), O(n^2), O(n) ?"

*log (n!) = Log (n)*(n-1)*(n-2)*...(1)
*logn! = Log(n) + Log(n-1) + Log(n-2) + Log(1)
*logn! = nLog(n)

Therefore:

O(log(n)) < O(n) < O(nlog(n)) < O(n^2)
